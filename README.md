## **📝 개요**
> ** 프로젝트 ** : 쇼핑 도우미
> ** 주제 ** : 무인 이동체를 이용해서 특정 위치에서 원하는 물품을 Cam을 통해 인식하여 물품을 받고 사용자에게 전달하기.
> ** 제작 기간 ** : 6.30 ~ 7.24
> ** 주요 기능 **
> > 받아야 하는 물품의 위치로 Mobile이동체 이동
> > Cam을 활용하여 Mobile 이동체의 인식및 물품 수량 배출
> > 받은 물품을 대기 위치로 이동



## **🛠 기술 및 도구**
> ** 언어 ** : Python, C/C++
> ** 환경 ** : Linux, ROS-melodic, Raspberry-pi, Arduino
> ** 라이브러리 ** : Qt, Tensorflow, OpenCv
> ** 도구 ** : Github
